# AI Coding Interview Assistant

---

# Project Objective

AI Coding Interview Assistant is a professional Android application developed using Kotlin and MVVM architecture.

The purpose of this application is to help students, freshers, and software developers prepare for coding interviews with the help of Artificial Intelligence.

The application provides an AI chatbot, resume analysis, code debugging, interview preparation, DSA practice, and coding assistance in one place.
